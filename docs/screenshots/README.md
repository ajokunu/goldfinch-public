# Screenshots

Drop PNG/GIF captures here and they render in the root `README.md`:

- `dashboard.png` — net worth, cashflow, weekly/monthly spending
- `budgets.png` — per-budget weekly/monthly/yearly periods
- `transactions.png` — list + the categorize/note detail sheet
- `investments.png` — holdings + the normalized price-history chart

A short GIF (e.g. `demo.gif`) of the dashboard and a categorize flow is the
highest-leverage single asset — it lets people see the app without deploying it.
Capture from the web export (`npx expo export -p web`) or a device build.
